# Lucasweb — Portfólio

Site portfólio profissional em React + Vite + Tailwind + Framer Motion.

**Live:** [lucasweb2025.github.io/lucasweb-portfolio](https://lucasweb2025.github.io/lucasweb-portfolio/)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Deploy (GitHub Pages)

1. Repositório: `lucasweb-portfolio`
2. **Settings → Pages → Build and deployment → Source:** selecione **GitHub Actions** (não "Deploy from a branch")
3. Push em `main` dispara o workflow — aguarde ~2 min em **Actions**

## Personalizar

- **WhatsApp / contato:** `src/data/site.ts`
- **Projetos:** `src/data/projects.ts`
- **Skills:** `src/data/skills.ts`
